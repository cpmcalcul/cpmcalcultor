## What is AIGXT?

AIGXT is an cpm calculator template powered by Next.js App Router, TypeScript, Tailwind CSS, Shadcn UI, Next-Auth, Next-Intl, Stripe, and Fumadocs. It helps you build production-grade AI applications quickly.
