# Installation

This guide will help you install and set up AIGXT on your local development machine.

## Prerequisites

Before installing AIGXT, make sure you have the following installed:

- **Node.js** (version 18.0 or higher)
- **pnpm** (recommended package manager)
- **Git** for version control

## Installation Steps

### 1. Clone the Repository

```bash
git clone https://github.com/your-org/aigxt.git
cd aigxt
```

### 2. Install Dependencies

```bash
pnpm install
```

### 3. Environment Setup

Copy the environment template and configure your variables:

```bash
cp .env.example .env.local
```

Edit `.env.local` with your configuration:

```bash
# Database
DATABASE_URL="postgresql://username:password@localhost:5432/aigxt"

# Authentication
NEXTAUTH_SECRET="your-secret-key"
NEXTAUTH_URL="http://localhost:3000"

# AI Services
OPENAI_API_KEY="your-openai-api-key"
REPLICATE_API_TOKEN="your-replicate-token"

# Stripe
STRIPE_PUBLISHABLE_KEY="your-stripe-publishable-key"
STRIPE_SECRET_KEY="your-stripe-secret-key"
```

```bash
pnpm dev
```

Your application will be available at `http://localhost:3000`.

## Verification

To verify your installation is working correctly:

1. Visit `http://localhost:3000`
2. Check that the homepage loads without errors
3. Navigate to `/docs` to see the documentation
4. Test the AI workstation at `/ai-workstation`

## Troubleshooting

### Common Issues

**Port already in use:**
```bash
# Kill process on port 3000
npx kill-port 3000
```

**Database connection issues:**
- Verify your `DATABASE_URL` is correct
- Ensure PostgreSQL is running
- Check database permissions

**Missing dependencies:**
```bash
# Clear cache and reinstall
rm -rf node_modules pnpm-lock.yaml
pnpm install
```
