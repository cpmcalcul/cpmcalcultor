# 安装指南

本指南将帮助您在本地开发机器上安装和设置 AIGXT。

## 前置要求

在安装 AIGXT 之前，请确保您已安装以下软件：

- **Node.js** (版本 18.0 或更高)
- **pnpm** (推荐的包管理器)
- **Git** 用于版本控制

## 安装步骤

### 1. 克隆仓库

```bash
git clone https://github.com/your-org/aigxt.git
cd aigxt
```

### 2. 安装依赖

```bash
pnpm install
```

### 3. 环境配置

复制环境模板并配置您的变量：

```bash
cp .env.example .env.local
```

编辑 `.env.local` 文件：

```env
# 数据库
DATABASE_URL="postgresql://username:password@localhost:5432/aigxt"

# 身份验证
NEXTAUTH_SECRET="your-secret-key"
NEXTAUTH_URL="http://localhost:3000"

# AI 服务
OPENAI_API_KEY="your-openai-api-key"
REPLICATE_API_TOKEN="your-replicate-token"

# Stripe
STRIPE_PUBLISHABLE_KEY="your-stripe-publishable-key"
STRIPE_SECRET_KEY="your-stripe-secret-key"
```

### 4. 数据库设置

初始化数据库：

```bash
# 生成迁移文件
pnpm db:generate

# 应用迁移
pnpm db:migrate

# (可选) 打开 Drizzle Studio
pnpm db:studio
```

### 5. 启动开发服务器

```bash
pnpm dev
```

您的应用程序将在 `http://localhost:3000` 上可用。

## 验证

要验证您的安装是否正常工作：

1. 访问 `http://localhost:3000`
2. 检查主页是否正常加载
3. 导航到 `/docs` 查看文档
4. 在 `/ai-workstation` 测试 AI 工作站

## 故障排除

### 常见问题

**端口已被占用：**
```bash
# 终止端口 3000 上的进程
npx kill-port 3000
```

**数据库连接问题：**
- 验证您的 `DATABASE_URL` 是否正确
- 确保 PostgreSQL 正在运行
- 检查数据库权限

**缺少依赖：**
```bash
# 清除缓存并重新安装
rm -rf node_modules pnpm-lock.yaml
pnpm install
```
